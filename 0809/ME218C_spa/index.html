Project Overview

This year's task is to build a remotely operated boat and a paired controller. The boats, in teams of four, need to navigate Terman pond to move a beachball into a goal. Both devices use Zigbee radio modules to communicate, and the class as a whole developed a communications protocol to make all devices interoperable and non-interfering.
